# 🔐 Mastermind (Go)

A web application to play the classic **Mastermind** code-breaking game against the computer, built in **Go**.

This is a Go port of the original [Rust implementation](../), preserving the identical REST API and reusing the same browser frontend.

## Features

- 🎮 **Classic Mastermind rules** – guess the secret 4-peg code (6 colors, 10 attempts)
- 🖥️ **Browser-based UI** – colorful, responsive, no external frameworks
- 🔍 **Post-game analysis** – minimax-based optimality scoring per turn
- 📤 **JSON export** – portable game log (`schema_version`, variant, all turns)
- 🐳 **Fully Dockerized** – single image, ready to run anywhere
- 🐹 **Pure Go** – chi router, stdlib structured logging, in-memory state

## Quick Start

### With Docker (recommended)

```bash
docker compose up --build
# → open http://localhost:8080
```

### From source

```bash
go run ./cmd/mastermind
# → open http://localhost:8080
```

The log level is controlled by the `LOG_LEVEL` environment variable (default: `info`).

## How to Play

1. Click **New Game** to get a secret code generated by the computer.
2. Click the empty circles to cycle through the 6 colors (Red, Orange, Yellow, Green, Blue, Purple).
3. Click **Submit** to check your guess.
4. ⚫ Black = right color, right position; ⚪ White = right color, wrong position.
5. Crack the code in 10 attempts to win!

After the game, click **🔍 Analyze My Game** to see per-turn optimality scores and suggested better guesses.

## REST API

| Method | Path | Description |
|--------|------|-------------|
| `GET`  | `/api/health` | Health check |
| `POST` | `/api/game` | Start a new game |
| `GET`  | `/api/game/{id}` | Get game state |
| `POST` | `/api/game/{id}/guess` | Submit a guess `{"guess":[1,2,3,4]}` |
| `POST` | `/api/game/{id}/analyze` | Analyze a completed game |
| `GET`  | `/api/game/{id}/export` | Export game log as JSON |

### Example

```bash
# Start game
GAME=$(curl -s -X POST http://localhost:8080/api/game)
ID=$(echo $GAME | jq -r .game_id)

# Guess
curl -s -X POST http://localhost:8080/api/game/$ID/guess \
  -H "Content-Type: application/json" \
  -d '{"guess":[1,1,2,2]}'

# Analyze after game ends
curl -s -X POST http://localhost:8080/api/game/$ID/analyze | jq .
```
